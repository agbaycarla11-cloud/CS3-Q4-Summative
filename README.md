# CS3-Q4-Summative
A minecraft themed shopping website.

A Minecraft-themed web development project created for the CS3 at Philippine Science Highschool. This project simulates an online trading outpost where players can browse categories, manage an inventory (cart), and complete trades.

## Project Structure
- `index.html`: The main landing page featuring a Minecraft-style menu.
- `category.html`: The core trading page with the product catalog and inventory form.
- `About.html`: Professional biographies and contributions of the development team.
- `style.css`: The primary stylesheet for the trading interface and global Minecraft UI components.
- `style1.css`: Specialized styling for the About page layout.
- `style2.css`: Dedicated styling for the Home page menu system.
- `images/`: Directory containing pixel-art assets (items like cookies, netherite, etc.).
